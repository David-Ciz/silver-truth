General goal of the project

Generating a new silver truth is part of a bigger project of MLEM-QA - Machine Learning using Ensemble Methods and Quality Assurance. The project aims to improve the quality of the silver truth. To achieve this goal, we want to explore two main directions: 1. QA part - Improving the selection criteria for the silver truth. 2. EM part - Improving the algorithm for generating the silver truth.
